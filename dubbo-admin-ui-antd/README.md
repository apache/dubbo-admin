# dubbo-admin-ui-antd

This project is initialized with Ant Design Pro v5. Follow is the quick guide for how to use.

## Environment Prepare

```bash
npm install
```

## Build Setup


### Start project

```bash
npm start
```

### Build project

```bash
npm run build
```

### Check code style

```bash
npm run lint
```

You can also use script to auto fix some lint error:

```bash
npm run lint:fix
```

### Test code

```bash
npm test
```
